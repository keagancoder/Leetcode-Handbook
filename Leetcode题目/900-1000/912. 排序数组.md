### 题目描述

给你一个整数数组 nums，请你将该数组升序排列。

**示例 1：**

```
输入：nums = [5,2,3,1]
输出：[1,2,3,5]
```


**示例 2：**

```
输入：nums = [5,1,1,2,0,0]
输出：[0,0,1,1,2,5]
```

**提示：**

- `1 <= nums.length <= 5 * 104`
- `-5 * 104 <= nums[i] <= 5 * 104`

---

### 解题思路：

#### 归并排序：

归并排序分为两个步骤，分别是**划分**和**合并**

- **划分：**按照中值划分左右两个数组直到左右两边都是一个
- **合并：**合并左右两个有序的part，这里引入暂存数组来保存替换的值
